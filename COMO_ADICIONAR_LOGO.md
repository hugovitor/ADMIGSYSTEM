# Como adicionar seu logo personalizado

O sistema está configurado para usar um ícone de igreja como logo placeholder. Para adicionar seu próprio logo:

## Opção 1: Substituir o logo SVG

1. Coloque sua imagem de logo em: `frontend/public/logo.png` (ou .svg, .jpg)
2. O logo aparecerá automaticamente no login e dashboard

## Opção 2: Usar um logo personalizado

Se você já tem um arquivo de logo, você pode:

1. Salvar o arquivo na pasta `frontend/public/` com o nome `logo.png`
2. O sistema detectará automaticamente

## Opção 3: Criar um logo com IA

Você pode usar ferramentas como:
- DALL-E
- Midjourney  
- Canva
- Figma

E exportar como PNG ou SVG para a pasta public.

## Tamanho recomendado

- Login: 100x100 pixels (mínimo 80x80 para mobile)
- Dashboard: 80x80 pixels (mínimo 60x60 para mobile)
- Formato: PNG com fundo transparente ou SVG

O sistema é totalmente responsivo e se ajustará automaticamente ao tamanho da tela!
